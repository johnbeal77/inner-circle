# Inner Circle

## Overview

Inner Circle is a modular platform for trusted groups such as families, friends, and roommates.

It enables private shared spaces called "Circles" where members can collaborate, play games, and use shared tools.

---

## Core Concept

Everything in the system exists inside an Inner Circle.

A Circle is a private group containing:
- members
- shared modules (tools and experiences)
- shared state and session data

---

## What a Circle Can Contain

- Household tools (Family Toolbox)
- Game experiences (Game Night)
- Planning tools
- Shared lists and coordination tools
- Future modules

---

## Design Principles

- Everything is scoped to a Circle
- Modules are plug-in systems, not hardcoded features
- Social trust is the foundation of all interactions
- The platform is modular and extensible

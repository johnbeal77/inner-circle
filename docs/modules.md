# Modules

## Overview

Modules are independent feature systems that plug into Inner Circle.

Each module is responsible for its own functionality but operates within the Circle context.

---

## Current Modules (Planned)

### Family Toolbox
A set of household coordination tools:
- grocery lists
- chores
- allowance tracking
- shared reminders

---

### Game Night
A social play module for group games.

Contains:
- session creation
- player invites
- Game Hat integration

---

### Future Modules
- Recipes & meal planning
- Event planning
- Media sharing
- Calendar coordination

---

## Module Rules

- Modules must operate within a Circle
- Modules must not manage users directly
- Modules must not bypass Circle permissions
- Modules are independent and reusable

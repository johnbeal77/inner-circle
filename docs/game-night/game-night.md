# Game Night

## Overview

Game Night is a social play module inside Inner Circle.

It allows Circle members to start shared game sessions, invite players, and launch game-related tools and experiences.

---

## Responsibilities

Game Night handles:
- session creation
- player invitations
- lobby flow
- session configuration
- launching game tools

---

## Game Hat

Game Hat is the first planned Game Night tool.

It supports prompt-based play such as:
- Act It Out
- Draw It

---

## Roles

### Host
Creates and controls the session setup.

### Player
A participant who accepted the session invite.

### Active Player
The player currently acting, drawing, or otherwise performing the prompt.

---

## Summary

Game Night is the orchestration layer for social play.

It does not contain all game logic directly. Game-specific rules belong in tools or engines such as Game Hat.

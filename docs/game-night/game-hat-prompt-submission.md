# Game Hat Prompt Submission

## Overview

Prompt submission is the process where Players contribute prompts to the shared Game Hat pool before gameplay begins.

This experience should feel:
- lightweight
- fast
- socially engaging
- easy to understand

---

## Core Principle

Submitting prompts should never feel like tedious data entry.

The process should encourage creativity while minimizing friction.

---

## Submission Flow

Typical flow:

1. Session enters prompt collection phase
2. Players receive prompt submission UI
3. Players submit prompts one at a time
4. System tracks completion progress
5. Session proceeds once requirements are satisfied

---

## Prompt Requirements

The Host configures:
- prompts required per Player

The system tracks:
- prompts submitted
- prompts remaining
- completion status

---

## Prompt Visibility

During submission:
- Players should only see their own prompts
- Players should not browse the full shared prompt pool
- Prompt creators remain tracked internally

---

## Editing Prompts

Preferred MVP behavior:
- Players may edit or delete prompts before session start

After gameplay begins:
- prompts become locked

---

## Submission UI Principles

The submission UI should prioritize:
- fast entry
- readability
- minimal navigation
- socially comfortable handoff on shared devices

---

## Shared Device Considerations

When using shared devices:
- prompt visibility should remain private
- quick handoff between Players should feel natural
- accidental prompt exposure should be minimized

---

## Completion Feedback

Players should clearly understand:
- how many prompts remain
- when they are finished
- when the group is waiting on others

Examples:
- progress indicators
- completion confirmations
- waiting status messaging

---

## Invalid Prompts

Possible invalid cases:
- empty prompt
- duplicate prompt (optional future rule)
- overly long prompt

Validation should remain lightweight and clear.

---

## Future Possibilities

Possible future additions:
- prompt suggestions
- themed prompt packs
- AI-assisted prompt generation
- saved custom prompt collections

---

## Design Principles

- low-friction creativity
- private prompt ownership
- lightweight validation
- socially comfortable interaction
- clear completion tracking
